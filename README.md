# GeoSphere Warn

Home Assistant Custom Integration für GeoSphere Austria Warnungen.

## Funktionen

- Warnungen nach GKZ
- Sensoren
- Binary Sensoren
- Config Flow
- Automatische Aktualisierung alle 5 Minuten

## Installation

1. Repository nach `custom_components/geosphere_warn` kopieren.
2. Home Assistant neu starten.
3. Einstellungen → Geräte & Dienste → Integration hinzufügen.
4. "GeoSphere Warn" auswählen.
5. GKZ eingeben (z. B. 31617).
